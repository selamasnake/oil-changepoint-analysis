# Notebooks

This folder contains Jupyter notebook used for exploratory data analysis (EDA)

- **eda.ipynb**: Initial exploration of the historical Brent oil prices with daily prices from May 20, 1987, to September 30, 2022.
- **brentoil_changepoint_analysis.ipynb**: Bayesian change point analysis of Brent oil log returns to detect structural breaks and associate them with key historical events.
